이 프로젝트는 opengl 라이브러리를 사용합니다. 따라서 glew, glm 등 라이브러리 설치가 필요할 수 있으므로 이 점 확인해주시면 감사하겠습니다.
pch.h에 헤더파일을 확인해 주세요!

본 프로젝트는 이전 OpenGL 수업 때 활용한 프로젝트를 기반으로 구현하였다는 점을 알려드립니다.

https://github.com/UihwanLee/ComputerGraphicsOpenGL

<사용시>
SERVER_ADDR = 127.0.0.1 입력 후 사용

<조작법>
p: 프로그램 종료 및 서버 접속 해제
w: 체스말 위로 이동
s: 체스말 아래로 이동
a: 체스말 완쪽으로 이동
d: 체스말 오른쪽으로 이동
